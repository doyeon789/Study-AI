# 선형 회귀 (Linear Regression)

## 개념 정리

- **선형 회귀** : 주어진 독립 변수(원인)와 종속 변수(결과) 간의 관계를 1차 함수 직선으로 모델링하여, 새로운 값을 예측하는 머신러닝 기법

- **가설(직선)** : $y = wx + b$

- **손실 함수(MSE)** : 예측값과 실제값의 차이를 제곱하여 평균낸 값. 이 값을 최소화하는 방향으로 학습

- **학습 방식** : MSE를 최소화하는 기울기(w)와 절편(b)을 찾는 과정. sklearn 내부적으로 정규방정식(Normal Equation) 또는 Gradient Descent 사용

## 실습 목표

가짜 데이터(`y = 3x + 5` + 노이즈)를 생성, 모델이 데이터를 보고 기울기(3)와 절편(5)을 잘 찾아내는지 확인.

## 실행 결과

| 항목                    | 값              |
| ----------------------- | --------------- |
| 학습된 기울기 (slope)   | 2.920 (정답: 3) |
| 학습된 절편 (intercept) | 5.286 (정답: 5) |
| MSE                     | 2.615           |
| R²                      | 0.969           |
