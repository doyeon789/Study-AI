# Hands-On ML3 Practice

이 저장소는 **『핸즈온 머신러닝 3판』**(오렐리앙 제롱 저, 박해선 역, 한빛미디어)의 실습을 따라 하며 머신러닝을 학습하기 위한 저장소입니다.

실습은 박해선님의 **handson-ml3** 저장소에서 제공하는 공식 예제와 노트북을 기반으로 진행했으며, 원본 자료는 [rickiepark/handson-ml3](https://github.com/rickiepark/handson-ml3)에서 확인할 수 있습니다.

## 📚 Study Contents

이 실습에서는 『핸즈온 머신러닝 3판』을 기반으로 머신러닝과 딥러닝의 핵심 개념부터 실제 모델 구축 및 배포 과정까지 학습합니다.

### Part 1. 머신러닝 기초 및 전통적 머신러닝

- 머신러닝 개념과 전체적인 학습 과정 이해
- End-to-End 머신러닝 프로젝트 workflow 경험
- 데이터 전처리 및 데이터 분석
- 모델 학습과 평가 방법
- 분류(Classification) 문제 해결

### Part 2. 머신러닝 알고리즘

- 선형 모델 (Linear Models)
  - 선형 회귀 (Linear Regression)
  - 로지스틱 회귀 (Logistic Regression)

- 서포트 벡터 머신 (Support Vector Machine)

- 결정 트리 (Decision Tree)

- 앙상블 학습 (Ensemble Learning)
  - 랜덤 포레스트 (Random Forest)
  - 여러 모델을 결합하는 방법

- 차원 축소 (Dimensionality Reduction)
  - PCA (Principal Component Analysis)

- 비지도 학습 (Unsupervised Learning)
  - K-Means Clustering
  - Gaussian Mixture Model
  - 군집 알고리즘 활용


### Part 3. 딥러닝 기초

- 인공 신경망 (Artificial Neural Network)
- TensorFlow와 Keras를 활용한 딥러닝 모델 구현
- 심층 신경망 (Deep Neural Network) 학습
- Custom Model과 Training Loop 구현
- 데이터 로딩 및 전처리 파이프라인 구성


### Part 4. 딥러닝 응용

- 합성곱 신경망 (CNN)
  - 이미지 분류
  - Computer Vision 문제 해결

- 순환 신경망 (RNN)과 CNN 기반 시퀀스 처리
  - 시계열 데이터 처리

- 자연어 처리 (NLP)
  - RNN 기반 NLP
  - Attention 메커니즘 이해

- 생성 모델
  - Autoencoder
  - GAN
  - Diffusion Model

- 강화학습 (Reinforcement Learning)
  - 에이전트와 환경의 상호작용
  - 보상 기반 학습

### Part 5. 머신러닝 시스템 구축

- 대규모 머신러닝 모델 학습 방법
- 모델 저장 및 배포 과정 이해
- TensorFlow Serving을 활용한 모델 서비스화


## 🛠️ Tech Stack

- Python
- Jupyter Notebook
- NumPy
- Pandas
- Matplotlib
- Scikit-Learn
- TensorFlow / Keras
